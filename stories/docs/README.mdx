# Documentation du site web

Bienvenue dans la documentation technique du site **wylow2ricard0.github.io**.

Ce guide a pour objectif d’accompagner les développeurs dans la prise en main, la contribution et la maintenance du projet.  
Vous y trouverez :

- Une présentation de l’architecture du site
- Les instructions pour installer et lancer le projet en local
- Les bonnes pratiques de développement et de contribution
- Les informations sur la configuration des outils (Next.js, Storybook, etc.)
- Un historique des modifications importantes

N’hésitez pas à compléter cette documentation à chaque évolution du projet afin de faciliter le travail de l’équipe.
