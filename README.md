# wylow2ricard0-portfolio

Portfolio personnel avec Next.js, déployé sur Vercel.

## Prérequis

- Node.js >= 18
- npm >= 9

## Installation

```powershell
npm install
```

## Développement

```powershell
npm run dev
# Ouvrir http://localhost:3000
```

## Build et production

```powershell
npm run build
npm run start
```

## Qualité de code

```powershell
npm run lint
npm run format
```

## Déploiement Vercel

- Branch de production: `main`
- Importer le repo sur Vercel
- Build: `npm run build` (auto)
- Variables: via Project Settings → Environment Variables

## Branches

- main: production
- dev: développement
