# feat(mui): installer Material-UI et dépendances

## Objectif

Installer Material-UI (MUI) v5 et ses dépendances pour créer une interface utilisateur moderne, responsive et accessible avec des composants React prêts à l'emploi.

## Raison du changement

- Composants UI professionnels et testés
- Design system cohérent (Material Design)
- Responsive et accessible par défaut
- Thème personnalisable
- Grande communauté et documentation complète
- Compatible Next.js et TypeScript

## Dépendances installées

- **@mui/material** : Composants Material-UI (Button, Card, Grid, etc.)
- **@emotion/react** : Moteur CSS-in-JS requis par MUI
- **@emotion/styled** : API styled pour composants personnalisés
- **@mui/icons-material** : Icônes Material Design (5000+ icônes)

### Pourquoi Emotion ?

MUI v5 utilise Emotion au lieu de JSS (v4) :

- **Performance** : CSS-in-JS plus rapide
- **Bundle size** : Plus léger que JSS
- **SSR** : Meilleur support Next.js
- **TypeScript** : Meilleur typage
- **API moderne** : sx props, styled API
