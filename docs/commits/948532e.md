## `package.json`

Manifeste du projet Node.js qui définit les métadonnées, dépendances et scripts.

### Scripts disponibles

```bash
npm run dev     # Démarre le serveur de développement (port 3000)
npm run lint    # Vérifie la qualité du code avec ESLint
npm run build   # Génère la version de production optimisée
npm run start   # Démarre le serveur de production
```

### Dépendances de production

| Package | Version | Rôle |
|---------|---------|------|
| `next` | ^14.0.0 | Framework React avec SSR, SSG et API routes |
| `react` | ^18.2.0 | Bibliothèque pour construire l'interface utilisateur |
| `react-dom` | ^18.2.0 | Package pour rendu React dans le DOM |

### Dépendances de développement

| Package | Version | Rôle |
|---------|---------|------|
| `eslint` | ^8.0.0 | Vérification de la qualité du code |

- **Node.js 18+** : Requis pour Next.js 14 et les dernières fonctionnalités
