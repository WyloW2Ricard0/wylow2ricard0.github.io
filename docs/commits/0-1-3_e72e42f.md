# feat(hero): implementer la section hero

## Objectif

Ajouter les polices, poser un thème de base et brancher la page d’accueil sur un premier composant Hero.

## Changements principaux

- Polices `@fontsource/righteous`, `@fontsource/roboto-mono`
- Fichiers thème :
  - `themes/palette.ts` : palette minimale (contrastThreshold)
  - `themes/typogrphy.ts` : typographie (Righteous pour titres, Roboto Mono pour le corps)
  - `themes/component.ts` : overrides de composants (Button, Stack, etc.)
  - `themes/theme.ts` : agrégation du thème
- Intégration app :
  - `pages/_app.tsx` : ThemeProvider + CssBaseline
  - `pages/index.tsx` : utilisation d’un composant `Hero`
  - `sections/hero.tsx` : section d’accueil avec Avatar, titres, boutons

## Fichiers créés/modifiés

- `.npmrc` : `legacy-peer-deps=true` (résolution des peer deps)
- `package.json` / `package-lock.json` : ajout des polices
- `pages/_app.tsx` : application du thème MUI
- `pages/index.tsx` : wrapper Container et insertion de `<Hero />`
- `sections/hero.tsx` : section hero (Avatar, titres, CTA)
- `themes/palette.ts`, `themes/typogrphy.ts`, `themes/component.ts`, `themes/theme.ts` : configuration du thème MUI
